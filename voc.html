Vocabulario Completo de Redes, Protocolos, Aparatos, Cables, Audio, Cámaras de Vigilancia y POS

A B C D E F G H I L M N O P R S T U V V

A

Access Point (Punto de Acceso)
Dispositivo que proporciona conectividad inalámbrica a una red. Ejemplo: Wi-Fi para cámaras de vigilancia.
AMQP (Advanced Message Queuing Protocol)
Protocolo de mensajería para aplicaciones distribuidas, soporta comunicaciones punto-a-punto y publicación/suscripción. Ejemplo: IoT industrial.
Audio over IP
Tecnología para transmitir audio digital a través de redes IP. Ejemplo: Sistemas de megafonía en red.

B

Backbone Cable
Cable principal que conecta diferentes segmentos de red, como fibra óptica en cableado vertical. Ejemplo: Interconexión de pisos en un edificio.
Bridge (Puente)
Dispositivo que conecta segmentos de red, filtrando tráfico por direcciones MAC. Ejemplo: Unir dos LANs en una red lógica.

C

Cable Coaxial
Cable con un conductor central rodeado por un blindaje, usado en CCTV analógico. Ejemplo: Conexión de cámaras analógicas a DVR.
Cable Ethernet (Cat5e, Cat6, Cat8)
Cables de par trenzado categorizados por velocidad y frecuencia. Cat5e soporta 1 Gbit/s, Cat6 hasta 10 Gbit/s, y Cat8 hasta 40 Gbit/s. Ejemplo: Redes de alta velocidad.
Cámara CCTV (Closed-Circuit Television)
Cámara analógica para sistemas de videovigilancia cerrados. Ejemplo: Sistemas tradicionales de seguridad en tiendas.
Cámara IP
Cámara que transmite video digital a través de una red IP, con su propia dirección IP. Ejemplo: Cámaras de vigilancia en red empresarial.

D

DDNS (Dynamic Domain Name System)
Servicio que actualiza direcciones IP dinámicas para mantener accesibles los dispositivos. Ejemplo: Acceso remoto a cámaras IP.
DDS (Data Distribution Service)
Protocolo para intercambio de datos en tiempo real, ideal para aplicaciones críticas como robótica o vehículos autónomos. Ejemplo: Control de tráfico aéreo.
DHCP (Dynamic Host Configuration Protocol)
Protocolo que asigna direcciones IP automáticamente a dispositivos en una red. Ejemplo: Configuración de IP en una cámara IP.
Disk Array (Matriz de Discos)
Sistema de almacenamiento redundante para datos en redes. Ejemplo: Almacenamiento de backups en una SAN.
DNS (Domain Name System)
Protocolo que traduce nombres de dominio a direcciones IP. Ejemplo: Convertir "google.com" en una IP numérica.
DVR (Digital Video Recorder)
Dispositivo que graba video de cámaras analógicas. Ejemplo: Sistemas de CCTV tradicionales.

E

EMV (Europay, Mastercard, Visa)
Estándar para transacciones seguras con tarjetas de crédito y débito. Ejemplo: Pagos con chip en terminales POS.

F

Fibra Óptica
Cable que usa luz para transmitir datos, ideal para largas distancias y alta velocidad. Ejemplo: Conexión backbone en redes empresariales.
Firewall
Dispositivo o software que protege la red filtrando tráfico según reglas de seguridad. Ejemplo: Bloqueo de accesos no autorizados a una red.
FTP (File Transfer Protocol)
Protocolo para transferencia de archivos entre cliente y servidor, independiente del sistema operativo. Ejemplo: Subida de archivos a un servidor web.

G

Gateway (Puerta de Enlace)
Dispositivo que conecta redes con diferentes protocolos, como TCP/IP y OSI. Ejemplo: Conexión de una LAN a Internet.

H

H.264/H.265
Codecs de compresión de video para reducir el tamaño de los archivos sin perder calidad. Ejemplo: Grabación eficiente en cámaras IP.
HTTP (Hypertext Transfer Protocol)
Protocolo para transferir archivos en la web, basado en un modelo cliente-servidor. Ejemplo: Carga de páginas web en navegadores.
HTTPS (HTTP Secure)
Versión segura de HTTP que utiliza cifrado TLS/SSL para proteger datos. Ejemplo: Transacciones en línea seguras.
Hub
Dispositivo simple que conecta múltiples dispositivos en una LAN, retransmitiendo datos a todos los puertos. Ejemplo: Conexión básica de PCs en una red pequeña.

I

IMAP (Internet Message Access Protocol)
Protocolo para gestionar correos electrónicos en el servidor sin eliminarlos, permitiendo acceso desde múltiples dispositivos. Ejemplo: Sincronización de correos en teléfonos y PCs.
IPsec (Internet Protocol Security)
Conjunto de protocolos para cifrar y autenticar datos en VPNs, operando en la capa de red. Ejemplo: Túneles VPN seguros.

L

LAN (Local Area Network)
Red de área local que conecta dispositivos en un área geográfica limitada, como una oficina o edificio, utilizando cables o conexiones inalámbricas. Ejemplo: Una red empresarial para compartir recursos como impresoras.

M

Mail Server (Servidor de Correo)
Servidor que gestiona el envío y recepción de correos electrónicos. Ejemplo: Servidor de email empresarial.
MAN (Metropolitan Area Network)
Red que cubre un área metropolitana, conectando varias LANs en una ciudad. Ejemplo: Redes de universidades o gobiernos locales.
Media Converter (Conversor de Medios)
Dispositivo que convierte señales entre diferentes tipos de cables, como cobre a fibra óptica. Ejemplo: Extensión de redes de vigilancia.
Micrófono IP
Micrófono conectado a una red para grabar o transmitir audio. Ejemplo: Audio en sistemas de videovigilancia IP.
Modem
Dispositivo que convierte señales digitales en analógicas y viceversa para conectar a Internet. Ejemplo: Conexión DSL o por cable.

N

NAS (Network-Attached Storage)
Dispositivo de almacenamiento conectado a la red que permite acceso centralizado a datos. Ejemplo: Servidor de archivos en una red doméstica.
NetFlow
Protocolo para recopilar metadatos sobre tráfico IP en routers y switches. Ejemplo: Análisis de tráfico de red en empresas.
NFC (Near Field Communication)
Tecnología para pagos sin contacto a corta distancia. Ejemplo: Pagos con teléfonos móviles en terminales POS.
NVR (Network Video Recorder)
Dispositivo que graba video de cámaras IP en una red. Ejemplo: Almacenamiento de grabaciones de vigilancia.

O

OmniSound HD
Tecnología que reduce el ruido de fondo en grabaciones de audio. Ejemplo: Mejora de calidad en conferencias telefónicas.
ONVIF (Open Network Video Interface Forum)
Estándar para la interoperabilidad de cámaras IP y sistemas de videovigilancia. Ejemplo: Integración de cámaras de diferentes marcas.

P

PAN (Personal Area Network)
Red de área personal para dispositivos cercanos, como auriculares inalámbricos o teléfonos móviles, con un alcance de unos 10 metros. Ejemplo: Conexión Bluetooth entre un smartphone y un smartwatch.
Patch Cable
Cable corto para conectar dispositivos en un rack o panel de conexiones. Ejemplo: Conexión de un switch a un router.
PCI DSS (Payment Card Industry Data Security Standard)
Estándar de seguridad para proteger datos de tarjetas en sistemas POS. Ejemplo: Cifrado de datos en transacciones.
PoE Switch (Power over Ethernet Switch)
Switch que suministra energía y datos a dispositivos a través de cables Ethernet. Ejemplo: Alimentación de cámaras IP.
POP3 (Post Office Protocol 3)
Protocolo para descargar correos electrónicos desde un servidor, eliminándolos del servidor tras la descarga. Ejemplo: Uso en clientes de correo como Outlook.
POS Network
Red que conecta terminales POS a servidores para procesar pagos. Ejemplo: Conexión de terminales a bancos para autorizar pagos.
PPPoE (Point-to-Point Protocol over Ethernet)
Protocolo para autenticar conexiones de red sobre Ethernet. Ejemplo: Conexiones DSL en hogares.
Print Server (Servidor de Impresión)
Dispositivo que gestiona trabajos de impresión en una red. Ejemplo: Impresión remota en oficinas.
PTZ (Pan-Tilt-Zoom)
Cámara con funciones de movimiento horizontal, vertical y zoom. Ejemplo: Cámaras de vigilancia en espacios públicos.

R

RTP (Real-time Transport Protocol)
Protocolo para transmitir audio y video en tiempo real. Ejemplo: Streaming de audio en aplicaciones VoIP.
RTSP (Real-Time Streaming Protocol)
Protocolo para controlar transmisiones de medios en tiempo real, usado en cámaras IP. Ejemplo: Streaming de video de vigilancia.
Router
Dispositivo que dirige el tráfico de red entre diferentes redes, como LANs y WANs. Ejemplo: Conexión de una red doméstica a Internet.

S

SAN (Storage Area Network)
Red dedicada para conectar dispositivos de almacenamiento, como matrices de discos, a servidores. Ejemplo: Almacenamiento centralizado en centros de datos.
SD-WAN (Software-Defined Wide Area Network)
Enfoque basado en software para gestionar redes WAN, optimizando el tráfico y la conectividad a la nube. Ejemplo: Soluciones de Cisco Meraki.
SMTP (Simple Mail Transfer Protocol)
Protocolo para enviar correos electrónicos. Ejemplo: Envío de emails desde un cliente de correo.
SNMP (Simple Network Management Protocol)
Protocolo para monitorear y gestionar dispositivos de red de diferentes fabricantes. Ejemplo: Supervisión de switches y routers.
STP (Shielded Twisted Pair)
Cable de par trenzado con blindaje para reducir interferencias. Ejemplo: Redes en entornos con mucho ruido electromagnético.
Switch
Dispositivo que conecta dispositivos en una red LAN, seleccionando rutas para datos. Ejemplo: Conexión de cámaras IP a un NVR.

T

TCP/IP (Transmission Control Protocol/Internet Protocol)
Conjunto de protocolos que permite la comunicación en Internet, dividiendo datos en paquetes y asegurando su entrega. Ejemplo: Navegación web y correo electrónico.
Terminal POS
Dispositivo para procesar pagos en puntos de venta, integrando hardware y software. Ejemplo: Terminal de pago en supermercados.
TLS (Transport Layer Security)
Protocolo de cifrado para proteger comunicaciones en red. Ejemplo: Seguridad en conexiones HTTPS.
TPV (Terminal Punto de Venta)
Sinónimo de POS, usado en entornos comerciales para transacciones. Ejemplo: Cajas registradoras electrónicas.

U

UDP (User Datagram Protocol)
Protocolo sin conexión, rápido pero menos confiable que TCP, ideal para streaming o videovigilancia en vivo. Ejemplo: Transmisión de video en tiempo real.
UPnP (Universal Plug and Play)
Protocolo para conectar dispositivos dinámicamente en una red, como cámaras o impresoras. Ejemplo: Configuración automática de cámaras IP.
UTP (Unshielded Twisted Pair)
Cable de par trenzado sin blindaje, usado en redes Ethernet. Ejemplo: Cableado de redes LAN Cat5e o Cat6.

V

VLAN (Virtual Local Area Network)
Red lógica que segmenta una red física en subredes, mejorando la seguridad y la gestión del tráfico. Ejemplo: Separar dispositivos IoT de la red corporativa.
VoIP (Voice over IP)
Tecnología para realizar llamadas de voz a través de redes IP. Ejemplo: Telefonía empresarial por Internet.
VPN (Virtual Private Network)
Tecnología que crea un túnel cifrado para conectar redes o dispositivos de forma segura a través de Internet. Ejemplo: Acceso remoto seguro a una red empresarial.

W

WAN (Wide Area Network)
Red de área amplia que conecta múltiples LANs a través de grandes distancias, como Internet o redes corporativas entre sedes. Ejemplo: Conexión de oficinas en diferentes ciudades.
