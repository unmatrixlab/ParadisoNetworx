Restaurador de memorias — Laura Editor

Este archivo reconstruye las memorias del editor usando los valores encontrados en la página final exportada.

Muy importante: para que funcione, abre este archivo desde el mismo origen donde usas el editor original. Si el editor corre en paradisonetworx.com, sube este HTML a ese dominio y ejecútalo allí. Si lo abres como archivo local y el editor está en un dominio web, Chrome no permitirá escribir en la misma memoria.

Qué va a restaurar:

  • Memoria 1 de Global, Menu, Header Menu, Hero, Music, Videos, Interviews, Sound, Featured, Coming y Footer.
  • El modo de Media Room detectado en la página exportada: all.
  • El orden de bloques base usado por el editor.
